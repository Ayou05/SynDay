# SynDay 产品与技术决策记忆

最后更新：2026-06-22（Asia/Shanghai）

## 产品定位

- 面向个人学习自律与情侣陪伴学习。
- 首批用户为两人，小范围自用，但工程按正式上架标准建设。
- iOS 与 Android 双端，不等待应用商店审核。
- 非必要不使用弹窗，不做高频催促和复杂长文。

## 平台与发布

- iOS：Apple Developer Program 签名，目标设备为 iPhone 17 Pro，iOS 27 beta 1。
- Android：签名 APK/AAB，目标设备为大陆版一加 Ace 5 / ColorOS；Windows MuMu 模拟器辅助测试。
- 应用名：SynDay 朝夕同序。
- Bundle ID / Android 包名：`cloud.catclaw.synday`。
- 域名：`catclaw.cloud`。
- API 域名：`api.synday.catclaw.cloud`。

## 后端与部署

- Go API 部署在腾讯云香港 2C2G Ubuntu 24.04。
- Supabase 使用新加坡区域。
- App 的核心业务请求尽量经过 Go API。
- Supabase 负责 Auth、PostgreSQL 和 RLS。
- GoEasy 负责在线情侣实时同步，不承担离线推送。
- AI 默认使用 DeepSeek，低延迟模型优先。

## 认证与账户

- 邮箱 + 密码登录。
- Supabase Auth 管理会话，Go 校验 JWT。
- Service Role 密钥绝不进入客户端。
- 支持账号删除：7 天冷静期后永久删除；该功能完整实现但不是首要链路。

## 时间与任务

- 固定北京时间 UTC+8，不支持跨时区配置。
- 每日 04:00 日切。
- 23:30 生成复盘草稿，04:00 锁定客观数据。
- 23:30–04:00 的新记录会刷新复盘统计。
- 任务计划支持单次、每日、每周指定星期。
- 计划时间可选，仅用于排序和复盘，不产生额外催促。
- 修改模板只影响未来任务，不修改历史实例。

## 专注

- 支持正计时、15/25/45/60 分钟和自定义倒计时。
- 进入专注不足 60 秒不算有效记录。
- iOS 不能强制开启系统专注模式，使用引导。
- Android 获得特殊权限后尝试控制勿扰，失败则降级引导。
- App 内进入沉浸界面并抑制自身非必要通知。

## 情侣绑定与可见性

- 主流程为“同序星图”扫码，本质使用一次性安全令牌。
- 6 位码和安全链接作为兜底。
- 双方确认后原子绑定，令牌 5 分钟失效且只能使用一次。
- 伴侣默认可查看完整当日任务标题和完成状态，但不可编辑。
- 双方不能查看对方复盘。
- 默认共享专注状态，保留单次“暂不共享”。

## 联机专注

- 允许一方先独自开始，另一方中途加入。
- 后加入者无需打断对方确认。
- 对方收到静默界面提示和普通系统通知。
- 双方各自记录独立开始、结束和专注时长。
- 系统额外计算两人重叠的共同专注时长。
- 一方退出、结束或断网不终止另一方。
- 计时以服务端 `started_at` 为准，不同步每一秒。
- 重连后从服务端状态恢复，重复消息不产生重复记录。

## 连胜与休息

- 每人独立计算个人连胜。
- 相伴连胜独立于个人连胜：当天任一方存在有效学习记录即可推进。
- 完成至少一条任务，或产生至少一条满 60 秒专注记录，即为有效学习。
- 固定休息日每周 1 天。
- 临时请假默认每月 4 天，最多连续 2 天；服务端策略可配置。
- 休息/请假日保护个人和相伴连胜但不增加天数；日历显示为豁免色，不伪装成学习日。

## AI

- AI 是增强层，永不阻塞核心数据写入。
- 任务激励在任务生成或创建后提前批量生成并缓存。
- 点击完成时只显示已经准备好的 AI 文案或本地兜底文案。
- 迟到的 AI 结果不替换用户已经看到的文案。
- 当前使用 `deepseek-v4-flash` 并显式关闭默认思考模式，避免短文案产生不必要延迟。
- 语气可选：克制温和、朋友陪伴、简短有力。
- 复盘数字由确定性代码计算，AI 只做中立分析、建议和语言整理。
- AI 失败时保存结构化复盘并后台重试。
- 不向模型发送邮箱、用户 ID、情侣身份、推送令牌或密钥。

## 通知

- App 完全关闭时仍要求接收情侣通知。
- iOS 使用 APNs。
- 大陆一加 Ace 5 使用 OPPO PUSH 主通道，FCM 备用。
- 所有通知可分类关闭。
- 各类别使用不同的短、轻、自然材质感声音。
- 使用普通通知，不申请关键警报，不绕过静音、专注模式或勿扰。
- 建议声音：
  - 复盘：柔和木鱼
  - 睡前：轻钟
  - 伴侣完成任务：短纸页
  - 伴侣加入专注：双音和弦
  - 连胜里程碑：明亮三音

## 弱网与离线

- 今日任务和计时状态保存在本地。
- 离线可完成任务、创建临时任务和结束计时。
- 恢复网络后通过幂等操作队列同步。
- 冲突由服务端版本号和时间裁决。
- 情侣实时状态和 AI 需要联网。
